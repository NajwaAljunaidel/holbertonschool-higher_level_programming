The ``0-add_integer`` module
============================

Using ``add_integer``
---------------------

This document demonstrates the usage of the ``add_integer`` function from the  
``0-add_integer`` module. Start by importing the function like this:

>>> add_integer = __import__('0-add_integer').add_integer

Example calls and their results:

>>> add_integer(1, 2)
3

>>> add_integer(100, -2)
98

>>> add_integer(4, 5.6)
9  # Floats are converted to integers before addition

>>> add_integer(4, "School")
